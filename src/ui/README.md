## 用户界面文档

### 核心组件

* `main_window.py` - 主窗口布局和交互逻辑

* **界面元素**：

  * 故事文本框 - 实时显示游戏叙事
  * 用户输入框 - 接受玩家输入的命令
  * 发送按钮 - 提交用户输入的命令
  * 图像显示区 - 展示当前场景的图像。
  * 音效播放 - 自动播放与故事相关的语音音效

### 交互流程

* 启动方式：运行`src/main.py`
* 输入方式：支持按钮提交和回车键响应
* 内容更新：提交后自动刷新故事、图像和音效

### 界面定制

* 使用`PyQt5`的样式表进行视觉定制
* 支持组件布局动态调整
* 提供响应式设计支持
